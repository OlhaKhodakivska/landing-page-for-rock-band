# 🎸 Thunder & Grit — Modern Rock Band Landing Page

A responsive and high-performance landing page for the fictional rock band **Thunder & Grit**.  
This project was built as a non-commercial educational exercise focused on modern frontend architecture, layout systems, and build tooling.

| Technology | Purpose |
|---|---|
| HTML5 | Semantic layout structure |
| CSS3 | Flexbox, Grid, animations |
| JavaScript (ES6+) | Interactivity, Fetch API, validation |
| Vite | Fast build tooling and dev server |

# ⚡ Features

- 📱 Fully responsive layout (mobile-first approach)
- 🍔 Custom mobile hamburger menu
- 🎛️ Interactive navigation system
- 📩 Contact form with real-time validation
- 🌐 Mock API integration for form handling
- 🎨 Smooth animations and transitions
- ⚡ Optimized assets via Vite bundling
- 🚀 Fast loading performance

# 🛠️ How to run locally

## 1️⃣ Clone repository

```bash
git clone https://github.com/olhakhodakivska/landing-page-for-rock-band.git
```

---

## 2️⃣ Enter project folder

```bash
cd landing-page-for-rock-band
```

---

## 3️⃣ Install dependencies

```bash
npm install
```

---

## 4️⃣ Start development server

```bash
npm run dev
```

The app will be available at:

```bash
http://localhost:5173
```

---

# 📦 Build for production

```bash
npm run build
```
