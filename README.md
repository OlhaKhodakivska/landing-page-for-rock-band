# 🎸 Thunder & Grit — Modern Rock Band Landing Page

A responsive, high-performance landing page for the fictional rock band **Thunder & Grit**. Built as a non-commercial educational project to practice modern frontend workflows.

## 🔗 [Live Demo](https://olhakhodakivska.github.io/landing-page-for-rock-band/)

## 🚀 Technologies Used

- **HTML5** (Semantic layout)
- **CSS3** (Flexbox, Grid, custom animations)
- **JavaScript** (ES6+, asynchronous Fetch API, form validation)
- **Vite** (Build tool & development server)
- **GitHub Actions/Pages** (Deployment)

## ⚡ Features

- **Fully Responsive:** Optimized for mobile, tablet, and desktop screens.
- **Dynamic Menu:** Custom hamburger navigation for mobile users.
- **Interactive Forms:** Contact form with real-time validation and mock API integration.
- **Optimized Assets:** Images and styles are bundled and minified via Vite for lightning-fast loading.
